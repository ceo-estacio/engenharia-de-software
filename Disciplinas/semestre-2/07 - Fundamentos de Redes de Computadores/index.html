O que é a internet?

redes de computadores
conjunto de módulos processadores interligados por um sistema de comunicação, capazes de trocar informações e compartilhar recursos.
internet
conjunto de rede de computadores que opera, utilizando os protocolos TCP e IP, e interconecta bilhões de dispositivos de computação

A internet é formada por

As três grandes partes

  • Borda da rede
  • Núcleo da rede
    ISPs ( global / regional )
  • Redes de acesso
    conecta os elementos de borda ao núcleo da rede

Modelo OSI

RMOSI da ISO: Reference Model For Open Systems Interconnection
Modelo de Referência Para Interconexão De Sistemas Abertos
É um modelo de 7 camadas, onde apenas as funcionalidadesde cada camada são muito bem definidas, e a maneira através da qual elas vão ser implementadas fica a critério de cada dev.
