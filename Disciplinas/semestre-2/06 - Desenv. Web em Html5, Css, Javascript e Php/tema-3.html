Módulo 4

Formulários em páginas Web

Tag <form>

atributos

action
Define a URL (endereço) para a qual os dados do formulário serão enviados.
method
Define o método HTTP (POST ou GET) para envio dos dados.
novalidate
não valida o form

atributos para inputs

placeholder
Exibir um texto no campo de input.
required
Determinar se um campo é obrigatório.
autofocus
Fixar o foco no campo.
pattern
RegEX - expressões regulares

pattern regex

pattern=“[0-9]{5}-[0-9]{3,4}$”

o DDD, com dois caracteres numéricos, seguido por dois conjuntos de números:
um contendo 5 e outro contendo 3 ou 4 caracteres.
