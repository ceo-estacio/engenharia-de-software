Princípios Fundamentais da Computação

reconhecer o processo de exploração do potencial dos computadores por meio de códigos e algoritmos

lógica do computador

Código

texto escrito em alguma linguagem de programação, para que a máquina compreenda o comando

Algoritmo

é o passo a passo das açoes que serão executadas pelo computador. Enquanto não estiver funcionando, não poderá ser classificado como um algoritmo, pois ainda não alcança sua finalidade.

Habilidades do programador

  • Decomposição

    dividir o problema em soluções menores

  • Abstração

    é a capacidade de olhar para um fenomeno no mundo real, retirar os detalhes que não são importantes para a resolução d o problema, e considerar apenas aqueles detalhes que realmente são importantes

  • Reconhecimento de padrões

    detectar que tipo de algoritmo pode ser usado para resolver o problema específico

  • Automação de Tarefas

    através de algoritmos, saber traduzir a solução em instruções para serem fornecidas ao computador

Questão

Sobre a diferença entre algoritmos e códigos de computadores, analise as afirmações abaixo e depois marque a opção que relaciona corretamente as afirmações a um código ou a um algoritmo.

  1. Sequência de passos simples e bem projetados para realizar uma tarefa ou resolver um problema.

    algoritmo

  2. Pode ser em uma linguagem de programação padrão.

    algoritmo

  3. Sequência de instruções que computadores podem executar.

    Código

  4. Deve ser escrito em uma linguagem de programação padrão, senão os computadores não conseguirão executá-lo.

    código

Questão

  1. O ser humano, que escreve códigos em alguma linguagem para/através de uma sequência de instruções bastantes simples, consegue gerar uma aplicação útil.

exercícios

  1. Grande parte da vida profissional de um programador é dedicada ao estudo de diferentes linguagens de programação e como elas podem atender a uma necessidade específica da melhor maneira possível. Acerca dos princípios fundamentais da computação, o que é um algoritmo?

    Um algoritmo é sequência finita de instruções simples e executáveis a serem seguidas para a solução de um determinado problema, especialmente por computadores.

  2. Para Aristóteles, o pensamento é a atividade da alma que permite ao ser humano compreender o conhecimento. Atualmente, o que é destacado na relação entre a capacidade de pensamento humano e a capacidade do computador?

    A capacidade de pensamento humano é infinita, enquanto a capacidade do computador é limitada.

  3. Os computadores são usados em uma ampla variedade de aplicações, incluindo processamento de texto, navegação na web, jogos, edição de vídeo, design gráfico e cálculo científico. Eles também são usados em muitas áreas da vida cotidiana, como compras online, bancos e serviços de saúde. Além disso, os computadores são fundamentais para a pesquisa e desenvolvimento de novas tecnologias. Essencialmente, o que é um algoritmo em um computador?

    Um algoritmo é uma série de instruções ou passos para solucionar um problema ou realizar uma tarefa.
