Como o programa vai parar na memória RAM?

  1. (load) copia as instruções da mem2 para a mem1
  2. a CPU faz o ciclo busca-execução para rodar o programa
  3. a CPU começa a rodar/executar o programa
  • Depois que uma CPU executa uma instrução, ela executa a próxima instrução na sequencia
  • "goto" Alterar a ordem de execução de instruções pela CPU, como, por exemplo, para implementar loops de repetição e estruturas condicionais.

Sistema Operacional

O sistema operacional está para os computadores como o governo está para uma nação.
SO: é uma interface entre o usuário e o computador
kernel: núcleo do SO, é onde ficam os drivers
boot: processo de carregamento do SO
ordem de carregamento:
  1. kernel
  2. drivers
  3. serviços
