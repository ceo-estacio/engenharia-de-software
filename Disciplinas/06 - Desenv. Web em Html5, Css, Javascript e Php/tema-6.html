Conceitos básicos de PHP

Escopo

todo script em php deve ser colocado dentro da tag <?php ... ?>

Variaveis

toda variavel no php começa com $

$name = "Nome"; $idade = 30;

Arrays

   <?php
      $colors = array( "red", "green", "blue" );
   ?>

Funções

   function addNumbers( $a, $b ) {
      return $a + $b;
   }

   function addNumbersStrict( int $a, int $b ) {
      echo $a;
      return $a + $b;
   }

   <?php declare( strict_types=1 ) // força o modo estrito
      function addNumbersStrict( int $a, int $b ) {
         return $a + $b;
      }
   ?>

Laços de repetição

   <?php
      for( $i = 0; $i <= 10; $i++ ) {
         echo "the number is: $i <br />";
      }

      foreach( $colors as $value ) {
         echo "$value <br />";
      } 

      $cars = array( "volvo", "toyota", "volkswagem" );
      echo count( $cars );
      echo shuffle( $cars );

      $bikes = [ "cicle", "motorcicle", "mountain bike" ];
      echo sort( $bikes );

      $idades = ( "Eu" => "39", "Ela" => "28", "Ele" => "8" );
      echo "nossa idade ".$idades[ "Eu" ]


   ?>

Conceitos básicos da linguagem

Trabalhando com formulários

  • $_GET
    $_GET[ "name" ] pega os dados transmitidos com get
    a informação é enviada no header
  • $_POST
    $_POST[ "name" ] pega os dados transmitidos através de post
    a informação é enviada em background
  • $_REQUEST
    $_REQUEST[ "name" ] coringa
