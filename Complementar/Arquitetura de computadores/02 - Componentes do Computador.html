os 4 Componentes Básicos do Computador

os 3 componentes principais do computador são interconectados por um barramento

  1. Processador

    execução de dados e intruções

  2. Armazenamento ( memórias principal e secundária )

    Memória Principal: RAM
    Armazenamento volátil.

    Memória Secundária: HD
    Armazenamento permanente.

  3. E/S

    dispositivos responsáveis pela entrada e saída de dados entre o computador e fora dele

    ex: teclado, mouse, impressora...

  4. Barramento

    responsável por interligar todos os componentes do computador

    ex: fios, trilhas, condutores ...
